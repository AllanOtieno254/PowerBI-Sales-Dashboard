## Description
This repository contains a Power BI project that demonstrates how to create a comprehensive sales dashboard. The dashboard includes key performance indicators (KPIs), visualizations, and measures to analyze sales data effectively.

### Features:
- Total Sales
- Sales by Category
- Top Product by Sales
- Sales Trends
- Average Sales per Unit

## Dataset
The `Dataset/sales_data.csv` file contains the following columns:
- **Product**: Name of the product sold
- **Category**: Category of the product
- **Sales**: Total sales amount
- **Quantity**: Number of units sold
- **Date**: Date of the sale

### Sample Data:
| Product   | Category     | Sales | Quantity | Date       |
|-----------|--------------|-------|----------|------------|
| Product A | Electronics  | 500   | 2        | 2024-12-01 |
| Product B | Furniture    | 300   | 1        | 2024-12-02 |
| Product C | Electronics  | 700   | 3        | 2024-12-03 |

## Dashboard
The Power BI dashboard (`Dashboard/SalesDashboard.pbix`) includes:
1. **Total Sales Card**: Displays total revenue.
2. **Sales by Category Bar Chart**: Visualizes sales distribution across categories.
3. **Top Product KPI**: Highlights the top-performing product.
4. **Sales Trends Line Chart**: Shows sales trends over time.
5. **Average Sales Gauge**: Indicates the average sales per unit.
